# Hermes Telegram Bot - Render Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/create/blueprint?env=TELEGRAM_BOT_TOKEN&repo=https://github.com/nivasvodapally/hermes-render-deploy)

## One-Click Deploy
1. Click the button above
2. Add your Telegram Bot Token
3. Done! Bot runs 24/7

## Manual Deploy
1. Fork this repo
2. Go to render.com → New → Web Service
3. Connect this repo
4. Add env var: `TELEGRAM_BOT_TOKEN` = your bot token
5. Deploy
